# Quick Task: Login and Signup Page Layout Optimization & Brand Logo Hover Fix

This quick task:
1. Modernizes and refactors the login and signup pages layout to fit all content on the screen without scrolling and center them inside a rounded card structure.
2. Removes default hover text underlines from the Syntra branding logo elements globally.

## Verification Criteria
- [ ] The login and signup pages do not have a vertical scrollbar on standard desktop screens (height >= 700px).
- [ ] On desktop, both pages feature a centered card with rounded corners, displaying branding on the left and the form on the right.
- [ ] The left branding panel uses a premium blue-to-indigo gradient background with white text.
- [ ] On mobile, the branding panel is hidden, and only the form is shown inside a responsive compact card layout.
- [ ] The Syntra branding logo does not display an underline when hovered on any page (Home, Login, Signup, complete_profile, and Dashboard).

## must_haves
- Wrap branding and form in a single `.auth-card` container.
- Reduce margins and paddings to fit the content within the viewport height.
- Set `text-decoration: none` on logo/branding links on hover globally.
